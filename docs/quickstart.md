# Quick Start

This guide walks through a complete example using bead to create a linguistic judgment experiment.

**Time to complete**: 15 minutes

## Overview

We'll create a forced-choice experiment testing argument structure alternations in English:
1. Define lexical items (verbs, nouns)
2. Create sentence templates
3. Generate experimental items
4. Partition items into balanced lists
5. Deploy to jsPsych/JATOS

## Prerequisites

Ensure bead is installed:

```bash
pip install bead
```
